# Weather App

Provides the users with an interface to interact and present the weather for any provided location
- By default, weather data is presented in the location's timezone, unless selected otherwise
- User can select to present weather data for a selected timezone from the drop-down list
- The application also presents a graph of the hourly forecast of the selected location

How To Run an Application

Application can be found under [Weather App](https://weather-urkwxqslzgelanr2ohm9dn.streamlit.app/)


To interact with the application

1.   Enter the city name in the input text area to get the weather forecast
2.   To present time in different time zones:
    1. Choose timezone
    2. Select the checkbox 'display date and time' in the timezone
3. Press the 'Present Weather' button
